# 6 Redis-Sorted Sets

Sorted Sets 就是一个有序的 Set, Set 中的每个元素都会有一个 score。

Sorted Sets 内部会按照元素的 score 进行排序, 当 score 相同时, 则通过 key 的 ASCII 码进行排序。

## 6.1 数据结构比较

| 数据结构 | 是否允许有重复元素 |  是否有序 | 有序的实现方式 |
| 列表 list  | 是  |  是 | 按照索引下标 |
| 集合 set | 否 | 否 | 无 |
| 有序集合 sorted sets | 否  | 是 | 按照 score 和 key 排序  |
