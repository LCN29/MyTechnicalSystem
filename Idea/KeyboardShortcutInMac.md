# Idea Keyboard Shortcut in Mac

## 1. 查看

### 类 

查看类中的方法
> command + f12, 出现下拉菜单
> command + 7, 左侧或者右侧会出现一个面板, 可以查看

### 类中的方法

查看方法引用
> 光标在方法上, command + B, 会出现下拉菜单
> 光标在方法上, option + f7, 底部会出现一个面板, 可以查看

搜索类
> command + O

## 4. 跳转

Tab 切换
